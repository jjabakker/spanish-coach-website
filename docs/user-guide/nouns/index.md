# Nouns Coach

The Nouns Coach lets you drill the most error-prone parts of Spanish noun vocabulary: **gender** (*el* vs *la*), **plural formation**, and **alternate-gender pairs** whose meaning depends on the article. Like Verbs, it is organised into **Setup**, **Study**, and **Test** sections.

---

## Setup

Before starting a session, choose which nouns are eligible:

- [Selecting Nouns](setup-nouns/) — filter by CEFR level, frequency, and topic category (home, food, body, technology, …).

---

## Study

Browse-only reference material — nothing is recorded.

- [Meaning Study](meaning-study/) — searchable list of all selected nouns with translations and articles.
- [Alternate Gender Explained](alternate-gender/) — guide to nouns whose meaning changes with the article (*el capital* / *la capital*, *el orden* / *la orden*).
- [Noun Gender Rules](noun-gender/) — reference notes on patterns that help predict whether a noun is masculine or feminine.
- [Plural Formation](plural-formation/) — the rules for forming Spanish plurals (the *-s* / *-es* / *-ces* patterns, and the unchanging-in-plural exceptions).

---

## Test

Active practice — every answer is recorded and feeds the adaptive engine.

- [Meaning Test](meaning-test/) — flashcards: Spanish word (with or without article) ↔ translation.
- [Pick the Correct Article](correct-article/) — choose *el* or *la* for each noun.
- [Singular / Plural](singular-plural/) — given the singular, produce the correct plural.
- [Alternate Gender Pairs](gender-pairs/) — practise nouns that change meaning with the article.
- [Recall Noun Gender](recall-gender/) — from the noun alone (no article), recall whether it is masculine or feminine.

---

## Recommended workflow

1. Start with **Select Nouns** at **A1**, Session Size **20**.
2. Run **Meaning Test** flashcards until the core vocabulary feels solid.
3. Add **Pick the Correct Article** to lock in the *el* / *la* distinction.
4. Use **Singular / Plural** once meanings and articles are confident.
5. The **Alternate Gender Pairs** drill is best left until later — it relies on already knowing the standard noun gender.
