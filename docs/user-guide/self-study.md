# Self Study

The Self Study screen lets you choose what type of vocabulary or grammar you want to practise.

<div class="screenshot-annotated">
![Self Study Screen](../assets/images/guide/self-study-screen.png)
</div>

1. **Back** — returns to the main screen
2. **Settings** — app and session preferences
3. **Verbs** — practise verb meanings, conjugation, gerund, participle, and irregular forms. [Learn more →](verbs-coach.md)
4. **Nouns** — practise articles, gender, singular and plural forms
5. **Words & Phrases** — practise adjectives, adverbs, and common expressions

---

## What's in each section?

**Verbs** is the most comprehensive section — it includes a full setup system for filtering which verbs and tenses you practise, plus six different test types. See the [Verbs Coach](verbs-coach.md) page for details.

**Nouns** and **Words & Phrases** follow a similar structure: select the words you want to study, then test yourself.
